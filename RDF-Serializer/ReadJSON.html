RDF Productontology Serializer is a Tool to serialize CSV and JSON product datasets in RDF syntax Turtle with a chosen vocabulary.

Transform your dataset in only 4 simple steps:

  1. Upload your CSV/JSON dataset. All dataset attributes will be extracted and listed by their quantity.
  2. Add the prefixes of the vocabulary you would like to use to describe the values of the dataset attributes.
  3. Define the ontology terms you would like to use to decribe the extracted dataset attributes.
  4. Finished! Download your Turtle file.
